# Disaster Management System 🚨🌍  
**A Smart Platform for Emergency Response & Disaster Relief**  

The **Disaster Management System (DisasterMS)** is a web-based solution designed to **coordinate rescue efforts, provide emergency services, locate resources, and facilitate donations** during crises. The platform ensures **quick response times**, **efficient resource allocation**, and **community engagement** for disaster relief.  

## 🌟 Features  
- 🚨 **Emergency SOS System** – Request immediate help during disasters.  
- 🏥 **Medical & Rescue Services** – Connect with first responders and medical teams.  
- 📍 **Resource Locator** – Find nearby shelters, food supplies, and medical aid.  
- 🤝 **Volunteer Portal** – Register as a volunteer and contribute to relief efforts.  
- 💰 **Donation Hub** – Support victims with monetary and material donations.  
- 📡 **Live Updates & Alerts** – Stay informed with real-time disaster reports.  

## 🚀 Technologies Used  
- **Frontend**: HTML, CSS (Tailwind CSS), JavaScript  
- **Frameworks & Libraries**: Alpine.js, Handlebars.js  
- **Tools**: ApexCharts.js, FontAwesome, Google Maps API  
- **Additional Features**: Responsive design, animations, real-time updates
