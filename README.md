# Disaster Management System

## 🌍 Overview
The **Disaster Management System** is a web-based platform designed to enhance disaster response and preparedness. It provides real-time alerts, emergency contact details, rescue coordination, and resource allocation tools to assist authorities and affected individuals during crises.

## 🚀 Features
- 🌎 **Real-time Disaster Alerts** – Get instant updates on natural and man-made disasters.
- 🆘 **Emergency Contacts** – Access critical emergency numbers and resources.
- 📍 **Live Location Tracking** – Helps authorities track affected areas and respond efficiently.
- 🏥 **Relief & Shelter Information** – Locate nearby relief camps and shelters.
- 📊 **Data Analytics & Reporting** – Visualize disaster data for better decision-making.
- 🔒 **User Authentication** – Secure access for volunteers and officials.

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS, JavaScript, Alpine.js
